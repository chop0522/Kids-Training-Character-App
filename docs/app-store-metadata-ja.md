# App Store メタデータ（日本語）

## アプリ名（App Name）
がんばりアルバム

## サブタイトル（Subtitle / 30文字以内）
習い事と勉強の成長記録

## コンセプト（1行）
親子で「できた！」を残して、成長を見返す記録アプリ。

## 説明文（Description案）
がんばりアルバムは、習い事や勉強の「できた！」を親子で一緒に記録して、あとから見返せる成長アルバムです。

・カレンダーで、いつ何をがんばったかがひと目でわかる  
・タグ（#リフティング #宿題 など）で、記録を整理＆検索  
・がんばるほど相棒が育って進化。宝箱のごほうびで続けやすい  
・勉強/運動など、家庭のルールに合わせて使えます

「今日は何ができた？」を、毎日の会話に。
小さな達成を積み重ねて、親子で成長を振り返りましょう。

## 使い方（箇条書き）
1. きょう：トレーニング（勉強/運動）を記録
2. きろく：カレンダーで日別に確認、タグで検索
3. ごほうび：宝箱を開けてモチベUP
4. あいぼう：育成＆進化、図鑑で思い出を残す

## キーワード候補（100文字以内の考え方・後で調整）
